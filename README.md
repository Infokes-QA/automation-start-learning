"# Automation Start Learning - Klinik Medis

Proyek Behavior-Driven Development (BDD) untuk automation testing sistem informasi pelayanan medis klinik menggunakan Cucumber.

## 📋 Deskripsi Proyek

Proyek ini mendefinisikan test scenarios menggunakan Gherkin untuk fitur-fitur utama klinik medis:
- **Pelayanan Poli Umum**: Dokumentasi pemeriksaan umum, diagnosis, dan resep obat
- **Pelayanan Poli Gigi**: Dokumentasi pemeriksaan gigi, tindakan, dan permintaan lab
- **Validasi Data**: Pengujian validasi kelengkapan data mandatory

## 📁 Project Structure

```
automation-start-learning/
├── tests/
│   └── features/
│       └── pelayanan.feature     # Feature file dengan 3 scenarios

```
